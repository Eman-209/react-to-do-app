# React Todo List App

A modern and responsive Todo List application built with **React** and **Tailwind CSS**. This project allows users to create, manage, and organize their daily tasks through a clean and intuitive interface. Tasks are stored in the browser using **localStorage**, ensuring they remain available even after refreshing or reopening the application.

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Eman-209/react-to-do-app.git
```

### 2. Navigate to the project directory

```bash
cd react-to-do-app
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

### 5. Open the application

Visit

```
http://localhost:5173
```

## 💡 How It Works

1. Enter a task.
2. Click the **Add** button.
3. The task appears in the list.
4. Mark a task as completed when finished.
5. Delete tasks you no longer need.
6. Tasks are automatically saved in the browser using localStorage.
